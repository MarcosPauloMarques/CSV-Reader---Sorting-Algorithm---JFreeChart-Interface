Este projeto utilizou a biblioteca "jfreechart-1.5.4" para a criação de interfaces. Esta biblioteca se encontra na pasta "lib" e é necessária para o funcionamento do projeto. Por favor insira ela á IDE ou o CMD para que o projeto funcione adequadamente.

A pasta "slides" contém os slides utilizados na apresentação.

A versão de Java utilizada para a criação deste projeto foi a versão "23.0.2".
